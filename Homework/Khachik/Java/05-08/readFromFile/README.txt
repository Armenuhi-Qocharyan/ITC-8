********Testing simulator*******

1) Description
    This program simulates programming testing process. You can enter numbers in input file  and then program runs in the output file you can see the  numbers multiplied by 10.

2) How to build.

    bash test.sh input.txt output.txt

3) Folder structure.

.
├── goutput.txt
├── input.txt
├── Main.java
├── output.txt
├── README.txt
└── test.sh
