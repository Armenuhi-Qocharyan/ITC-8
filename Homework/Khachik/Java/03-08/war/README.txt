**********WAR**********

1) Description
    This programm  simulates war. You can create soldiers of varius ranks. You can place them on the meadow. You can move them. You can shoot at the other soldiers. Soldiers of different ranks have different possibilities.

2) How to build
    jacac Main.java
    java Main
