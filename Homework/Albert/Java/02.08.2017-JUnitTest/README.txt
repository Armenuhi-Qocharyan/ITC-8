1) Description.
    In the battlefield there must be soldiers who will shoot each other in a given direction. They have weapons,count of bullets and coordinate.

2)Dependency
    junit-4.XX.jar          Download link:   https://github.com/junit-team/junit4/releases
    hamcrest-core-1.3.jar   Download link:   http://search.maven.org/remotecontent?filepath=org/hamcrest/hamcrest-core/1.3/hamcrest-core-1.3.jar

3) How to run Tests
    cd DIR
    bash run.sh

2) How to build.
    $ javac Main.java
    $ java Main
